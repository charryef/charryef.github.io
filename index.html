Blocitoff

Blocitoff is the first backend specialization that I chose for my Full Stack Web Development course at Bloc. It is an application, built with Ruby on Rails, that allows users to keep their to-do lists manageable by creating self-destructing ones. If the to-do item is not important enough to be completed in seven days, it gets automatically deleted.

Blocmarks

Blocmarks is my second backend specialization project that I chose for my Full Stack Web Development course at Bloc. It is an application that allows users to bookmark URLs via email, manage and share them with friends.
