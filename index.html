Blocitoff

Blocitoff, built with Ruby on Rails, is an application that allows users to keep their to-do lists manageable by creating self-destructing ones. If the to-do item is not important enough to be completed in seven days, it gets automatically deleted.


Technologies: Ruby on Rails, Devise, Faker, Ajax, Bootstrap
